A simple WordPress plugin that allows to protect pages and posts and assign them to specific users.

== Description ==

If you want to start selling ebooks, online courses or other digital contents,
you may need a protected area where securely share your contents.

This is the right plugin to start building your membership site.
It's small, fast and easy to use. Just install it, create your premium contents and protect them in a single click!

== Installation ==

You can easily install this plugin in your WordPress site following these steps:

1. Download the ZIP archive.
2. Upload the protect-content folder into your WordPress plugins folder.
3. Activate the plugin.
4. Create a post or a page you want to protect
5. In the "Protect Content" box, activate the "Protect" checkbox.
6. Assign the users you want to be able to access the content.
7. Optionally, specify a redirect url where send logged users that don't have access to the content.
8. Enjoy!
